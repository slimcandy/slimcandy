Start displaying texts the ‚Äúcorrect‚Äú way. Use hyphens, right dashes, hanging punctuation and quotation. A few lines of css make text book-like looking.

## Code

```css
blockquote {
  /* Safari only hanging punctuation */
  hanging-punctuation: first force-end last;
  font-variant-ligatures: normal;
  /* First load native font, than custom one */
  text-rendering: geometricPrecision;
  /* Book-like hyphens */
  hyphens: auto;

  /* Native quotes */
  quotes: '\00AB''\00BB''\201E''\201C';
}

blockquote:before {
  /* Native quotes */
  content: open-quote;
}
blockquote:after {
  /* Native quotes */
  content: close-quote;
}
```
