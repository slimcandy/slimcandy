## Source Code

```html
<form>
  <fieldset>
    <legend>Contact details</legend>
    <dl>
      <dt>
        <label for="email">E-mail</label>
      </dt>
      <dd>
        <input id="email" maxlength="255" minlength="5" type="email" />
      </dd>
    </dl>
  </fieldset>
</form>
```
