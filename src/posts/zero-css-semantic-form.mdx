W3C people spent months to create wide-range semantic html. I respect them and use proper tags for proper solutions. Below you see how omit a single div making semantic and good looking form.

## Source Code

```html
<form>
  <fieldset>
    <legend>Contact details</legend>
    <dl>
      <dt>
        <label for="email">E-mail</label>
      </dt>
      <dd>
        <input id="email" maxlength="255" minlength="5" type="email" />
      </dd>
    </dl>
  </fieldset>
</form>
```

## Real example
