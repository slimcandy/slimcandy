My very basic setup to start any project.

1. [Install Create React App with Typescript](https://create-react-app.dev/docs/adding-typescript/)

   `npx create-react-app my-app --template typescript`

2. ESLint is already in CRA, no need to
   [install it](https://eslint.org/docs/user-guide/getting-started).

   Initialize it - `npm init @eslint/config`

3. [Install Prettier](https://prettier.io/docs/en/install.html)

   `npm install --save-dev --save-exact prettier`

   - Integrate it into eslint (I prefer
     [standard way](https://prettier.io/docs/en/integrating-with-linters.html)) -
     https://github.com/prettier/eslint-config-prettier#installation

     `npm install --save-dev eslint-config-prettier`

   - Configure _.eslintrc.json_

4. Install Airbnb ESLint configs. It is
   [made for JS](https://github.com/airbnb/javascript/blob/master/packages/eslint-config-airbnb-base/rules/imports.js#L139).

   Install
   [_eslint-config-airbnb-typescript_](https://github.com/iamturns/eslint-config-airbnb-typescript)
   to bind it with typescript.

   `npm install --save-dev eslint-config-airbnb-typescript`

   - Configure _.eslintrc.json_
   - Some rules are not clear for me now, so I look up and escape them.

     For example
     [_jsx-a11y/label-has-associated-control_](https://github.com/jsx-eslint/eslint-plugin-jsx-a11y/issues/302#issuecomment-425512505)

   ```json
   "rules": {
      "jsx-a11y/label-has-associated-control": [ "error", {
        "required": {
          "some": [ "nesting", "id"  ]
        }
      }],
      "jsx-a11y/label-has-for": [ "error", {
        "required": {
          "some": [ "nesting", "id"  ]
        }
      }]
    }
   ```

5. Add scripts to _package.json_

```json
{
  "format": "prettier --write src/**/*.{ts,tsx}",
  "lint": "eslint --fix src/**/*.{ts,tsx}",
  "test": "npm run format && npm run lint"
}
```

6. Install [husky](https://typicode.github.io/husky/#/?id=automatic-recommended)

   `npx husky-init && npm install`

7. Profit!!! No more packages or settings are required.
