## Set Up

---

- [How to Set Up a Mac for Web Development](https://michaeluloth.com/how-to-set-up-a-mac-for-web-development)

### Node

- Install [Homebrew](https://docs.brew.sh/Installation)
- Install [`n`](https://www.npmjs.com/package/n) via brew to omit `node`
  duplicates (Apple M1)
- Use `sudo n` to choose `node` version

### NPM

- Latest: `npm install npm@latest -g`
- Specific version: `npm install npm@6.14.14 -g`

### Terminal

Install [Oh My Zsh](https://github.com/ohmyzsh/ohmyzsh)

### Chrome extensions

- [React Developer Tools](https://chrome.google.com/webstore/detail/react-developer-tools/fmkadmapgofadopljbjfkapdkoienihi?hl=en)
- [JSON Viewer](https://chrome.google.com/webstore/detail/json-viewer/gbmdgpbipfallnflgajpaliibnhdgobh)
- [WhatFont](https://chrome.google.com/webstore/detail/whatfont/jabopobgcpjmedljpbcaablpmlmfcogm?hl=en)
  or use FireFox
- [Octotree](https://chrome.google.com/webstore/detail/octotree-github-code-tree/bkhaagjahfmjljalopjnoealnfndnagc)

### Safari extensions

- [JSON View](https://apps.apple.com/ru/app/json-peep-for-safari/id1458969831?l=en&mt=12)
- [Octotree](https://apps.apple.com/ru/app/octotree/id1457450145?l=en&mt=12)

### Git

`ssh-keygen` - to make a key pair and link to bitbucket/gitlab/github.
[HOWTO: Set up an SSH key](https://support.atlassian.com/bitbucket-cloud/docs/set-up-an-ssh-key/).
[Troubles](https://support.atlassian.com/bitbucket-cloud/docs/troubleshoot-ssh-issues/)

#### Lint

Install [Prettier](https://prettier.io/docs/en/install.html) and
[ESLint](https://eslint.org/docs/user-guide/getting-started).

_.prettierrc.json:_

```json
{
  "printWidth": 80,
  "tabWidth": 2,
  "semi": false,
  "singleQuote": true,
  "trailingComma": "none",
  "bracketSpacing": true,
  "arrowParens": "avoid",
  "proseWrap": "always"
}
```

### Tips

"engines": {
  "npm": ">=8.5.1",
  "node": ">=16.4.0"
}
